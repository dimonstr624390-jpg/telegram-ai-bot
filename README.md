Программа работает, как телеграмм бот, которая может различать языки программирования, выдавая ответ, как сообщение.
(ФАЙЛЫ-КАРТИНКИ, ПОЖАЛУЙСТА, ИГНОРИРУЙТЕ)

ИНСТРУКЦИЯ:
1. Сделайте снимок вашего экрана, на котором запечетлён ваш код.
2. Отправьте снимок в телеграм бота
3. Подождите
4. Бот отправит вам ответ, всё готово
